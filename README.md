# Smart Exam Scheduling System

##  Description
This project implements an exam scheduling system using the Greedy Graph Coloring Algorithm. It assigns time slots to subjects while avoiding conflicts between exams.

##  Concept
- Subjects → Nodes
- Conflicts → Edges
- Time Slots → Colors

##  Algorithm Used
Greedy Graph Coloring

##  Features
- Conflict-free scheduling
- Optimized time slot usage
- Simple and efficient implementation

##  Sample Output
Time Slot 1:
- Maths
- Chemistry
- Biology

Time Slot 2:
- Physics
